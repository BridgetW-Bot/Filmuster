Filmuster is a movie review site whose main aim is to allow Kenyan users to rate and review Kenyan films & TV Shows.
Embedded videos include trailers and interviews with actors.
Intergration of Machine Learning algorithms allows curation of personalized recommendations based on a user's watch list.
The site also has various interactive features such as trivia, polls, movie themed ontests, etc
Users are able to discuss with each other on the platform.
